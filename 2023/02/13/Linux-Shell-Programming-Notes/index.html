字符串比较测试

比较两个字符串变量的值是否相等，不等这样的情况

= 判断是否相等

!= 判断是不等的

! 取结果的反义

注意

  • 对于字符串变量的比较一定要记住给变量加双引号
  • 使用等于号判断值是否相等，左右两边也必须有空格 # [ “yuzhang” = “yuzhang” ] && echo ok || echo no
